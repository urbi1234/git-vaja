\#Git day



\##Comande

-git init -> ustvari prazen repozitorij v trenutni mapi

-git remote add origin "url" -> nastavi origin trenutnega repozitorija na url oddaljenega repozitorija

-git status -> pokaže status sprememb datotek od prejšnjega commita

-git add file -> doda fajl gitu za spremljanje

-git commit -m "sporocilo" -> uveljavi sremembe ki so bile addane lokalno z napisanim sporocilom

-git push origin branch -> potisne spremembe na oddaljen repozitorij za vejo branch

-git pull origin branch -> potegne spremembe iz oddaljenega repozitorija iz veje branch na lokalen repozitorij

-git checkout -b branch -> naredi novo vejo branch in se premakne nanjo

-git checkout branch -> premakne se na vejo branch

-git branch -> izpiše vse veje in na kateri veji smo

-git merge branch -> združi vejo branch v trenutno vejo
